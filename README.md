####  运行配置环境说明

```shell
# python 虚拟环境
yum install ipython

# 安装 py自动补全库
pip install  jedi elpy  rope flake8 importmagic autopep8 yapf

```





Go emacs 配置文件

```shell
# go的自动补全库
go get -u github.com/nsf/gocode

# 开启联想 默认false
gocode set propose-builtins true

M-x package-install RET go-autocomplete

```
